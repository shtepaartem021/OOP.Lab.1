## Тема проекту  
Робота з класами в С#

## Мета проекту  
Поглибити знання про класи в C#, навчитися проєктувати систему з кількох взаємопов'язаних класів. Практикувати принципи композиції об'єктів та розподілу відповідальності між класами на прикладі симуляції дуельного клубу.

## Опис  
- Реалізовано основну структуру класів для симуляції дуелі: класи для заклять, ефектів заклять, чарівників (wizards), механізмів дуелі тощо.  
- Код написаний мовою C#; проєкт містить рішення (.sln), вихідні файли (.cs) та виконувану консольну програму.  
- Приклад: клас `Wizard` має набір заклять, яке може застосувати; клас `SpellEffect` — опис ефекту заклять; логіка дуелі реалізована у відповідному класі.

## Як запустити  
1. Відкрити рішення `DuelingSimulation.sln` у середовищі розробки, що підтримує C#.  
2. Зібрати (Build) проект.  
3. Запустити програму (наприклад, `Program.cs`).  
4. У консолі буде симуляція дуелі згідно з логікою всередині програми.

## Структура каталогу  
/ — корінь проєкту
DuelingSimulation.sln — рішення
Program.cs — точка запуску
Wizard.cs — клас чарівника
Spell.cs, SpellEffect.cs — класи заклять та їх ефектів
DuelingClub.cs, DuelResult.cs — логіка дуелі та результати
